# 🚀 Node.js + Express + SQLite Signup Form

A simple full-stack web application that allows users to create an account using a form.  
The data is sent to a Node.js backend and stored in a SQLite database.

---

## ✨ Features

- User signup form (First name, Last name, Email, Password)
- Frontend form validation
- Password show/hide toggle
- Fetch API communication (frontend → backend)
- Express server handling API requests
- SQLite database storage
- Clean UI with separate CSS file
- Modular project structure

---

## 🧱 Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express.js
- Database: SQLite
